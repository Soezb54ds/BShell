Steps to run BShell:
1) Have python 3.10 or greater installed in your system.
2) Make sure you can make a bat file of your own to start the terminal.
3) Also, mak sure python can be run in your command prompt.
4) Then, you can clone this repository.
5) Finally, if needed, make a system environmeent variable to make sure you can run it without sourcing the python file everytime. (Optional)
